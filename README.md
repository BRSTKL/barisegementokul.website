# Portfolio Energy Engineer

Personal portfolio website built with React, TypeScript, Vite, Tailwind CSS, Framer Motion, and shadcn/ui components.

## Projects

### Smart Order Tracker

Real-time order management dashboard concept for Siemens Energy Gas Services. It includes automated delay detection, KPI monitoring, regional analytics, sortable order tracking, and priority-based alerting.

### Construction Portfolio Data Lab

Interactive construction portfolio intelligence demo for ingesting Excel and PDF data, running QA checks, exporting reports, and generating secure AI-backed summaries.
